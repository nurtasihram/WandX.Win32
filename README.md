# WandX.Win32

`WandX.Win32` 是從目前解決方案中扇出的 Win32 模組原始碼目錄，已整理為可獨立版本控制的 Git 專案。

## 內容

此目錄包含 `WandX.Win32` 相關的 `.cppm`、`.h`、`.idl`、`.inl` 與進入點原始碼。

## 目前狀態

此專案已可作為獨立 Git 儲存庫管理，但原始碼目前仍依賴上層 `WandX` 模組，例如 `import WandX;`，因此尚未保證可完全脫離原解決方案單獨建置。
